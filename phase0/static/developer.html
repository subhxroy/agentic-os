Quickstart

1. Install SDK

# Python
pip install agentos

# TypeScript
npm install agentos

2. Authenticate

# Python
from agentos import AgentOS
client = AgentOS(base_url="http://localhost:8000")
client.register("you@example.com", "your-password")

# TypeScript
import { AgentOS } from "agentos";
const client = new AgentOS({ baseUrl: "http://localhost:8000" });
await client.register("you@example.com", "your-password");

3. Chat with the Agent

# Python
response = client.chat("What is the meaning of life?")
print(response.response)

# TypeScript
const response = await client.chat("What is the meaning of life?");
console.log(response.response);

4. Upload Knowledge

# Python
doc = client.upload_document("./my-policy.txt")
results = client.search_knowledge("What is the refund policy?")

# TypeScript
const doc = await client.uploadDocument(fileBuffer, "policy.txt");
const results = await client.searchKnowledge("What is the refund policy?");

5. Build a Custom Plugin

# plugin.py
def get_weather(city: str) -> str:
    """Get current weather for a city."""
    return f"Weather in {city}: Sunny, 22C"

# manifest.json
{
  "name": "weather",
  "version": "1.0.0",
  "description": "Weather plugin",
  "author": "you",
  "tools": [{"name": "get_weather", "description": "Get weather", "parameters": {"city": {"type": "string"}}}]
}

API Reference

Authentication

POST /api/auth/register Register new user
POST /api/auth/login Login, get JWT token

Chat

POST /api/chat Send message to agent
GET /api/sessions List chat sessions

Knowledge

POST /api/knowledge/upload Upload document (multipart)
POST /api/knowledge/search Semantic search

Developer

GET /api/developer/apps List your apps
POST /api/developer/apps Create app
POST /api/developer/keys Create API key
GET /api/developer/usage Usage analytics
GET /api/developer/openapi.json OpenAPI 3.0 spec

Plugins

GET /api/developer/plugins List published plugins
POST /api/developer/plugins Create plugin
POST /api/developer/plugins/:id/install Install plugin

GDPR & Enterprise

POST /api/gdpr/export Request data export (ZIP)
POST /api/gdpr/delete Request data deletion

Plugin Development

Plugin Manifest

{
  "name": "my-plugin",
  "version": "1.0.0",
  "description": "Does something useful",
  "author": "you",
  "tools": [
    {
      "name": "my_tool",
      "description": "A useful tool",
      "parameters": {
        "input": { "type": "string", "description": "The input" }
      }
    }
  ],
  "hooks": ["on_message", "on_response"],
  "config_schema": {
    "api_key": { "type": "string", "required": true }
  }
}

Tool Code

# plugin.py
def my_tool(input: str) -> str:
    """Process the input and return a result."""
    return f"Processed: {input}"

def on_message(message: str) -> str:
    """Hook: called before agent processes message."""
    return message  # pass through, or modify

Testing

from developer.plugins import PluginLoader, PluginSandbox

loader = PluginLoader()
manifest = loader.load_plugin("./my-plugin/")
tool = loader.get_tool("my_tool")

sandbox = PluginSandbox(timeout=5)
result = sandbox.execute_tool(tool, {"input": "hello"})
print(result)  # {"success": True, "result": "Processed: hello"}
