# Agentic OS Brain

This is the Obsidian vault used as the persistent memory brain for the Agentic OS.

## Structure

- `memories/` - Agent memories, facts, and user preferences (auto-synced from Agentic OS memory)
- `conversations/` - Session summaries and conversation logs
- `skills/` - Learned skills and capabilities
- `journal/` - Daily activity logs and reflections
- `projects/` - Active project notes and context
- `templates/` - Note templates for consistent formatting

## How It Works

Agentic OS syncs memories to this vault as markdown files. Each memory becomes a
searchable, linkable note in Obsidian. The agent can read and write to this vault,
creating a bidirectional knowledge graph between the AI and your notes.
